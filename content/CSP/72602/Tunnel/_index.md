72602 隧道统一采用双入口反向隧道：

- 主入口：`10021`
- 备入口：`10022`
- 目标 ECS：`47.110.67.161 (ecs-99)`

快速连接命令：

```bash
ssh -p 10021 aaron@47.110.67.161
ssh -p 10022 aaron@47.110.67.161
```

上线顺序建议：

1. 先在 72602-minipc 创建并启动 `10022`（备入口）
2. 验证 ECS 已监听 `10022`
3. 再创建并启动 `10021`（主入口）
4. 最后做外网双端口连通性验证
